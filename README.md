# Genial 3D Impressões

Landing page estática para vender um colecionável 3D com CTA direto para WhatsApp.

## Stack

- HTML
- CSS
- JavaScript
- GitHub
- Vercel

## Objetivo

Apresentar 1 produto com:

- headline forte
- prova visual
- benefícios
- FAQ
- CTA direto para WhatsApp

## Publicar na Vercel

1. Entre na Vercel
2. Importe este repositório
3. Framework preset: `Other`
4. Deploy

Como é um projeto estático simples, não precisa build complexa.
